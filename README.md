# Stock Market Dashboard with AI Sentiment Analysis

## Frontend:
- Framework: React with TypeScript
- Libraries: Chart.js or D3.js for visualizations, Tailwind CSS for styling

## Backend:
- Framework: Node.js with Express for API handling
- Sentiment Analysis: Python with Flask/FastAPI or integrate via pre-trained models (e.g., Hugging Face Transformers)

## Database:
- MongoDB for user preferences, historical stock data, and sentiment analysis results
- Redis for caching frequently accessed data

## Data Sources:
- Alpha Vantage/Finnhub for stock data
- News API/Twitter API for sentiment analysis inputs
